# Game-trading-system
基于Spring Boot与Vue的游戏交易系统，集成了游戏商品发布、浏览、购买、支付及售后等功能。后端采用Spring Boot，保障交易数据的安全与业务逻辑的高效处理；前端使用Vue.js，打造流畅且用户友好的界面。该系统旨在打造一个安全、便捷的游戏交易平台，让玩家能够轻松买卖游戏道具、账号等商品。同时，系统提供完善的支付与售后服务，确保玩家的交易体验与权益，促进游戏交易市场的健康发展。
